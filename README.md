Monte Carlo Path Tracer
======================

University of Pennsylvania, CIS 561: Advanced Computer Graphics, Course Project
------------
# Features:
- Ray Intersection
- Direct Lighting/global illumination
- Normal Map
- Multiple Importance Smampling
- Lens-based Camera
- Acceleration: Kd-Tree
- Normal Mapping
